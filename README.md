# vim setting
vimの設定ファイル群

# 説明
vimを快適に使えるように設定を行うファイルを作成し、インストーラーとして提供する。

### 機能
1. 行番号表示
2. タイトル表示
3. カラースキーム
4. インデント
5. カーソルの変更
6. 文字コード
7. 不可視文字の可視化
8. Beep音の無効化
9. プラグインのインストール
