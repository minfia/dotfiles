# dotfilesのインストール
各dotfileを管理し、それらをインストーラとして提供する  


## 対象
* vim
* tmux
* shell
  * bash


### 各README
* [vim](vim_setting/README.md)
* [tmux](tmux_setting/README.md)
* [shell](shell_setting/README.md)


## インストール方法
インストールする設定をオプションで指定する  
ex ) vim + shell(bash)  
```shell
$ ./install.sh --vim --shell bash
```

必要があれは、helpを確認すること
