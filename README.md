# dotfilesのインストール
各dotfileを管理し、それらをインストーラとして提供する

* tmux
* shell
  * bash
* w3m
* git

## インストール方法
インストールする設定をオプションで指定する  
ex ) vim + shell(bash)  
```bash
$ ./install.sh --vim --shell bash
```

必要があれは、helpを確認すること
